There are manly 7 type of topology 
0. Point to Point | (Many Book not includes this topology) many in PAN  point to point connection to node
1. Bus | All the nodes are connect though one wire it is simple but if a wire cut other nodes can't communicate 
2. Ring | All the nodes are connect in a ring it is simple but the devices can't cross communicate they have to relay other node
3. Star | All the nodes are connect to a one node or router it is simple but if the main router will fail the connection will not possible [LAN]
4. Mesh | All the nodes are inter connected it is costly because of the wires but very reliably 
5. Tree | Big university use this node as connect as parent and child like a family tree.
6. Hybrid | This is hybrid of all the above topologies internet many on this toplogies
