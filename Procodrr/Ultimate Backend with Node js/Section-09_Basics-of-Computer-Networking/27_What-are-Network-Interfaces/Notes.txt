5/3/2026 ✅What are Network Interfaces?
Notes:- 
Network Interfaces
Devices which is used to access internet
It is minimum 4 ways in mobile and Computer
Computer
1. Wifi or we use wifi card
2. internet is way to use a lan cable and connect to our computer internet port (Speed depends on Cable Quality)
3. Bluetooth use Bluetooth as data transfer between mobile and pc vie enabling Bluetooth  tethering 
4. USB is used between phone and computer 

Speed Comparison
Internet Port > USB Cable > Wifi > Bluetooth

In Phone
1. Wifi
2. Bluetooth
3. USB
4. Cellular Data (SIM Card)
